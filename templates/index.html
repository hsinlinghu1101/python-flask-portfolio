Sermo

Sermo is an app for learning new languages using spaced-repetition. App provides the user with a total of 10 words in Mandarine language one word at a time requesting the user to type the English translation. The words appear more often when guessed incorrectly, and appear less often when guessed correctly. Users can create an account and keep track of their progress. It’s for users who want to learn the Mandarin language. --HTML, CSS, Javascript, React, Node.js, Express, PostgreSQL, Cypress
